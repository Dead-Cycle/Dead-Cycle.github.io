算术,字符串与变量

2.1 数字和表达式

交互式python解释可以直接当作非常强大的计算器使用. 例如

解释器会给出答案4. 绝大数情况下,它的算术运算功能与计算器相同,但整数除法有些例外.

解释器给出的答案是0! 发生了什么? 一个整数被另一个整数除,计算结果的小数部分被截取了,只留下了整数部分. 有些时间这个功能很有用,但通常更多的情况是需要普通的除法.
对此,有两种解决方案: 1. 使用实数(带有小数部分)进行除法运算. 实数在python中被称为浮点数(Float), 执行**>>>1.0/2.0**,将会得到0.5; 2. python改变除法的执行方式. >>> from __future__ import division 执行这条语句,python只执行普通的除法. (此时,单斜线/不再是整除作用,python提供了一个用于实现整除的操作符//双斜线, 例如:>>> 1//2)
此外,计算机中还有一些重要的运算符:

取余运算符%, >>> x%y结果是x除以y的余数.
幂(乘方)运算符** , >>> 2**3=8

2.2 长整形数

python可以支持并处理非常大的整数, 并对其进行运算.

2.3 浮点数

在使用小数点方面,python非常灵活. 比如3.等价于3.0,.5等价于0.5
区分整数和浮点数很重要,比如5和5.0, 因为它们的内部表示大相径庭, 另外与整数不同的是,浮点数存在上限和下限, 超出上限或下限将导致溢出. 溢出意味着计算结果太大或太小,python无法将其表示为浮点数.

2.4 数学函数和导入模块

python自带了很多编写好的代码模块, math是其中之一. 要使用模块math或其他python模块都必须先将模块导入:

这样就可以访问math库中的数学函数了. 例如 math.sqrt(5)
下面是另一种导入模块的方式. 使用导入方式from module import *,但注意如果函数与math模块中的某个函数同名, 将被math模块中的同名函数覆盖. 因此使用import module通常更安全, 因为不会覆盖任何函数.

另外,还可以导入模块的特定函数,例如, from math import sqrt, 将只导入函数.
在导入模块后, 可使用函数dir(module) 列出模块的所有函数

字符串即一系列字符, 如何标识字符串, 在python中,可使用下列三种方式

  • 单引号, 如’http’
  • 双引号, 如”http”
  • 三引号, 如 “”” http“”” 三引号可用于多行字符串

它们的一个主要用途是,能在字符串中包含字符’和” 例如:

  1. 字符串长度: **len(s)**函数可以返回字符串包含多少个字符
  2. 字符串拼接: 可以将字符串进行相加+, 拼接多次可以使用*, 3*”ha”-> “hahaha”

在python中,数字和字符串的一个重要特征是不可变,即不能以任何方式修改,对于看似修改的情况下,python实际上是在创建修改版本的拷贝.

2.6 类型转换

隐式转换: python会自动在数值类型之间进行转换
显式转换:

  • 整数和字符串转换为浮点数: float(x)函数 如 float(3) float(“3”)
  • 整数和浮点数转换为字符串: str(n)函数 如 str(85) str(-9.79)
  • 浮点数和字符串转换为整数: int(x)函数 如 int(8.64) int(‘5’)
    此外还有round(x)函数, 可对整数圆整, 但要注意可能向下圆整, 如round(8.5)-> 8.5 .
    python模块math还提供了很多对数字进行处理的函数: math.trunc(x) math.ceil(x) math.floor(x)

2.7 变量和值

变量实际就是代表(或者引用)某值的名字, 通俗理解与数学上的变量无异.

此操作称为赋值(assignment),变量x此时的值为3.在变量被赋予值之后,就可以在表达式中使用该变量.(在使用变量之前,需要对其进行赋值,也被称为初始化操作). 对于x = expr这样的赋值语句,可这样解读: x只是指向表达式expr的值.赋值语句并不会复制指向的值, 而只是标记和重新标记既有值. 因此, 无论变量指向的对象有多大,多复杂,赋值语句的效率都非常高.

python一般会跟踪所有的值,对于没有任何变量指向的值,python会自动将其删除,这被称为垃圾收集.

在python中,有一种便利的技巧,让你能够同时给多个变量进行赋值.

字符串

6.1 字符串索引

字符串中的字符可通过索引去访问, 索引分正索引和负索引, 正索引由0开始,指向字符串第一个字符, 负索引由-1开始, 指向字符串最后一个字符.

6.2 字符串切片

在python中,可使用切片从字符串中提取子串. s[begin:end]返回从索引begin到end-1的子串,begin默认为0, end默认为字符串末尾. 执行切片操作也可以使用负数索引

  • 字符串的函数
函数名 返 回 值
find(t) 如果找到字符串,返回索引,否则返回-1
rfind(t) 与find相同,由右往左搜索
capitalize() 将首字母转换成大写,其余转为小写
lower() 所有字母转为小写
upper() 所有字母转为大写
swapcase() 所有字母大小写相互转换

strip(ch) 默认情况,删除空白字符,如指定字符串,则删除指定字符
partition(t) 以t为分隔,将字符串拆分为(head,t,tail)三部分
rpartition(t) 同partition,但由右往左搜索

数据结构

在python中, 主要的数据结构是列表和字典.列表按顺序存储数据,而字典使用键高效地存储和检索数据.使用type命令可获取数据的类型

7.1 序列

序列有3种内置的类型: 字符串,元组和列表, 特征是可使用切片表示法来复制子序列, 可用函数len计算其长度, 使用表达式 x in s来检查序列s是否包含x元素, 使用+,*进行拼接.

7.2 元组

元组是一种不可变序列, 包含0个或多个值. 由()括起,其中的元素用逗号分隔(1,2).
空元组表示为(), 但一个元素的元组表示方法比较特殊, 用**(x,)**表示. 如果省略逗号,就不是创建元组,而是表达式.

元组函数

函数名 返 回 值

len(tuple) 元组的长度
count(x) 元素x在元组中出现的次数
index(x) 元组中第一个x元素的索引,如果不包含x元素,引发异常

7.3 列表

列表与元组的区别是, 列表是可变的, 由[]括起. 列表中的元素指向相应的值,包括列表本身,这会形成自引用,Python能够识别自引用.

remove(x) 删除第一个x元素
pop(i) 删除并返回索引为i的元素
reverse() 反转排列顺序

7.3 字典

字典是python中一种存储键-值效率极高的数据结构. python字典中的每个键值都会通过散列函数转换为一个数字—-散列值. 字典的值被存储在一个底层列表中, 其索引为散列值. 通过这样来达到高效率访问.

语法

  1. 定义字典:{ "key" : "value”}
  2. 获取key对应的值, 使用 dict[key], 或者dict.get(key)

字典函数

函数名 返 回 值
items() 返回字典的视图
keys() 返回由字典key值组成的视图
values() 返回由字典value组成的视图
get(key) 返回与key相关联的值
pop(key) 弹出与key相关联的值,并删除
popitem() 弹出字典中的某个键值对,并删除
clear() 删除字典中的所有元素
copy() 复制字典
fromkeys(s, t) 创建一个新的字典,其中键来自s, 值来自t
setdefault(key, v)
update(e) 将e中的键值对更新到字典中, e可能是字典,也可能是键值对

一些实用的Python包

  • PIL : python图像处理库,支持众多图像格式,可用于执行裁剪,大小调整,旋转和滤镜效果等

python2.x与3.x区别

  1. 在python2.x下,请求输入使用raw_input, python3中input代替了raw_input的功能,但无法对字符串求值, 可用eval(input(expr))模拟代替.
  2. print函数, 在python2.x中,print从技术上来说并非函数,而是语言一部分,因而可以不用括号,但对于修改默认分隔符和结束字符串, 不如3.x方便.
