使用docfx工具自动生成C#的API文档

使用 docfx 构建您的技术文档站点。将 .NET 程序集、XML 代码注释、REST API Swagger 文件和 markdown 转换为呈现的 HTML 页面、JSON 模型或 PDF 文件。

创建API网站

  1. 首先需要安装.NetSDK 6.0或者更高版本

  2. 安装.NETSDK后, 打开终端并输出入以下命令安装最新的docfx

  1. 等待命令行安装docfx后, 添加docfx所在目录到环境变量中

4.对dll文件使用docfx生成文档集

5. 另一个命令在指定目录docfx_project创建一个新的文档集
